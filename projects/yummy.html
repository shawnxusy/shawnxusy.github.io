A WIKIPEDIA REDESIGN

Project Overview

Yummywiki is everything Wikipedia - but cleaner and smarter. Apart from a different layout system and varied color schemes, Yummywiki has implemented two additional functions: wiki toolbar, and knowledge graph.

Date: November 2014
Duration: 2 weeks
Team: Shawn Xu
Role: Design and implementation

{ 15 }

The number of tools used to design and hack Yummywiki

Process showcase

The project took a very agile approach, where designing and programming interweave in many of the iterations. The overall flow was: laying out the main pages (multiple iterations) -> building parser for Wikipedia -> designing and writing knowledge graph -> designing and writing wiki toolbar -> minor fixes

  • The very first iteration of overall layout, with suggestions from peers

  • Final, implemented iteration of site layout, with a few specifications

  • Programmatic flow for using the website

Project Outcome

As the first version of site was launched, a few user testing were conducted. The feedback gathered mainly focused on maneuverability of the knowledge graph.
I am also considering adding "parellel" viewing enabling users to view multiple articles side by side at the same time.
